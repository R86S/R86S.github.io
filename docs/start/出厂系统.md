## 出厂系统

### 基础介绍

目前出厂系统是符合中国法律法规的 合规 纯净版本Openwrt系统

可以进入官方售后QQ群下载重新安装的出厂系统

### 接口介绍
通用信息:

|ETH0|ETH1|ETH2|
|  ----  | ----  | ----  |
|千兆网卡(WAN)|2.5G(LAN)|2.5G(LAN)|

万兆版本:

|SFP+|SFP+|
|  ----  | ----  |
|万兆光口(未分配)|万兆光口(未分配)|
